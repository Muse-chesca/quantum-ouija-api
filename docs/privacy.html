Quantum Ouija – Privacy Policy

Last updated:

Overview

Quantum Ouija is a custom GPT that calls my API at quantum-ouija-api.onrender.com to generate messages using quantum-random numbers. This policy explains what data is sent, how it’s used, and your choices.

What data is processed

  • From ChatGPT to my API: the GPT sends only what’s needed for the request (parameters like max_symbols, etc.). It does not include your identity from ChatGPT.
  • Server logs (standard): my hosting provider (Render) may record IP address, timestamps, status codes and basic request metadata in access/error logs for reliability and abuse prevention.
  • Third-party QRNG: the server calls the ANU QRNG endpoint to fetch random bytes. That request contains no user-identifying content.

How data is used

  • To generate the quantum message you asked for.
  • To keep the service secure and working (rate-limits, debug via provider logs).

Retention

I do not intentionally store request bodies. Hosting provider logs may be retained for a limited period per their policy and then expire automatically.

Sharing & selling

No selling of data. No sharing with marketers. Data may flow to service providers strictly to run the service (Render for hosting; ANU for random numbers).

Security

The API requires an x-api-key. Please don’t share your key. Transport uses HTTPS.

Children

This tool is for general audiences but is intended for adults. If you believe a minor has provided personal information, contact me to request deletion.

Your choices

  • Don’t use the GPT if you’re uncomfortable with the above.
  • You may request removal of any persistent logs that could identify you (to the extent I control them).
